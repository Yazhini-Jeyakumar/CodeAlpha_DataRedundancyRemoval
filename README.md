# CodeAlpha Data Redundancy Removal System

## Overview
A Flask-based web application that prevents duplicate data from being stored in the database.

## Features
- Duplicate Email Detection
- Duplicate Phone Detection
- Data Validation
- Search Records
- Delete Records
- Export Records to CSV
- Responsive User Interface

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Project Objective
To improve database accuracy and efficiency by identifying and preventing redundant data entries.

## How to Run
1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000
